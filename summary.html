MIS 201 Course Summary

Chapter 7: Telecommunications, the Internet, and Wireless Technology

This chapter explores the fundamental components, architectures, and emerging trends in wired and wireless networking. It is structured around four core learning objectives:

  • Identify the principal components of telecommunications networks and key networking technologies.
  • Distinguish among different types of networks.
  • Explain how the Internet and Internet technologies function to support communication and e-business.
  • Describe the principal technologies and standards for wireless networking, communication, and Internet access.

1. Network Components and Key Networking Technologies

A computer network connects two or more computers via hardware (network interface cards, hubs, switches, routers) and software (network operating systems).

Hubs broadcast incoming packets to all connected devices; switches and routers manage traffic more intelligently by directing packets only to their intended destinations.

Client/server computing has largely replaced centralized mainframes, with servers managing communication rules and clients requesting services over a network—the Internet being the largest implementation of this model.

Packet switching slices messages into packets sent via multiple paths and reassembled at the destination, maximizing bandwidth efficiency.

TCP/IP is the four-layer protocol suite (network interface, Internet, transport, application) that underpins all Internet communication.

2. Types of Networks and Transmission Media

Networks are categorized by geographical span and signal type:

  • LANs (Local Area Networks) cover up to 500 meters, typically within a building.
  • MANs (Metropolitan Area Networks) extend across cities.
  • WANs (Wide Area Networks) span regions, countries, or continents; the Internet is the most universal WAN.

Transmission media include twisted-pair wire, coaxial cable, fiber optics, and wireless (satellite, cellular). Bandwidth and transmission speed vary by medium, measured in bits per second (bps).

3. The Internet: Architecture, Addressing, and Governance

The Internet is a global network of networks, with access provided by Internet Service Providers (ISPs). Connection options range from dial-up (56.6 Kbps) to DSL, cable, fiber, and satellite, up to tens of Mbps.

Every device has a 32-bit IP address; the Domain Name System (DNS) maps human-readable domain names to IP addresses via a hierarchical set of DNS servers.

Backbone networks are owned by Tier-1 providers; regional and local ISPs connect businesses and consumers.

Governance and standards are managed by bodies such as the IAB, ICANN, and W3C.

4. Internet Services and Communication Tools

Key Internet-based services include:

  • E-mail, instant messaging, and newsgroups for text and file exchange.
  • FTP and Telnet for remote file transfer and system access.
  • World Wide Web services using HTTP, HTML, and URLs to retrieve hyperlinked content.
  • Voice over IP (VoIP) and unified communications integrate voice, video, and data over IP networks.
  • Virtual Private Networks (VPNs) create secure, encrypted tunnels over public networks.

5. The Web, Search, and Web 2.0/3.0

The Web is built on hypertext, enabling interconnected documents via embedded links.

Search engines (e.g., Google’s PageRank) index and rank content; semantic search and social search refine results using language understanding and social signals.

Web 2.0 emphasizes user-generated content, interactivity, and collaboration through blogs, wikis, microblogging, and social networks.

Web 3.0 envisions a more intelligent, pervasive web—integrating the Internet of Things (IoT), cloud-based SaaS, and seamless connectivity.

6. Wireless Technologies and Standards

Cellular Systems have evolved from 3G (144 Kbps) to 4G/LTE/WiMax (up to 100 Mbps), supporting mobile voice and data.

  • Wi-Fi (802.11) provides WLAN and Internet access via access points; hotspots offer public coverage but often lack strong security.
  • Bluetooth (802.15) enables short-range personal area networking among up to eight devices.
  • WiMax (802.16) extends wireless coverage up to 31 miles using specialized antennas.
  • Radio Frequency Identification (RFID) uses tags and readers for asset tracking (toll collection, supply chains).
  • Wireless Sensor Networks (WSNs) comprise numerous low-power devices to monitor environments, generating “big data” and fueling IoT applications.

7. Case Studies and Interactive Sessions

The chapter includes real-world cases and class discussions to illustrate applications and ethical considerations:

  • Telepresence moving beyond boardrooms into field operations.
  • Virtual collaboration with tools like IBM Sametime.
  • Dundee Precious Metals using underground Wi-Fi and RFID to enhance mining efficiency.
  • Debates on net neutrality and employee monitoring explore policy, legal, and business impacts.
  • Future directions with IPv6 (expanding address space) and Internet2 (100 Gbps research network).
